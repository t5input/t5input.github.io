# T5筆畫輸入法線上版

🌐 **線上版本**
- https://t5input.pages.dev

T5筆畫輸入法線上版，提供方便、免安裝的筆畫輸入法、字典和教學，讓您輕鬆使用並查詢筆畫輸入法。

## ✨ 主要功能

### 📝 輸入法工具
- **簡易版輸入法** - 快速、輕量的筆畫輸入介面
- **完整版輸入法** - 功能完整的筆畫輸入系統
- **輸入法字典** - 查詢字詞的筆畫編碼
- **輸入法教學** - 學習筆畫輸入法的使用方法

### 🛠️ 實用小工具
- **QR Code 產生器** - 快速生成 QR Code
- **螢幕錄影工具** - 瀏覽器內錄製螢幕畫面（支援系統音訊 + 麥克風）
- **MP3 切割工具** - 線上切割 MP3 音訊檔案

### 🌍 多語言支援
- 繁體中文（預設）
- English
- 簡體中文（透過 OpenCC 轉換）

## 🚀 特色功能

### 螢幕錄影工具
- ✅ 免安裝，直接在瀏覽器中使用
- ✅ 支援系統音訊錄製
- ✅ 支援麥克風錄音
- ✅ 音訊混合（系統音 + 麥克風）
- ✅ 暫停/繼續錄影
- ✅ 即時預覽
- ✅ WebM 格式輸出

### MP3 切割工具
- ✅ 自動分割（固定時間間隔）
- ✅ 手動指定範圍切割
- ✅ 完全在瀏覽器端處理（無需上傳）
- ✅ 支援 ZIP 打包下載
- ✅ 獨立下載每個片段
- ✅ Web Worker 多執行緒處理

### QR Code 產生器
- ✅ 即時生成 QR Code
- ✅ 自訂大小與顏色
- ✅ 支援多種資料類型
- ✅ 下載為圖片
